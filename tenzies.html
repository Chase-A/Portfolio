Tenzies

Background

Buiding the game Tenzies is part of the Scrimba front end career path cirriculum. I didn't want to follow the project exactly as it was laid out, so I learned Tailwind CSS and used it to style the app.

Development

While Building the app, I grew comfortable the useState() and useEffect() React hooks to implement the game's logic. I used inline Javascript in the JSX to conditionally render the instructions paragraph based on screen size, and confetti from the react-confetti library if the player wins the game.

GitHub

Result

The result of my efforts is a responsive game of tenzies that plays well on both desktop and mobile.
